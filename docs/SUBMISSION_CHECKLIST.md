# Submission checklist

## Product and evidence

- [x] Public mobile-first MVP.
- [x] Deterministic replay that works without live match availability.
- [x] Server-only TxLINE connector with graceful failure states.
- [x] Public source repository and technical documentation.
- [x] AI assistance disclosure and human-decision record.
- [x] Unit, end-to-end, responsive, and accessibility checks.
- [x] Automated demo recording, narration, H.264/AAC rendering, and captions.
- [x] Activate the free TxLINE devnet tier.
- [x] Verify a covered historical fixture through the deployed connector.
- [x] Render and publish the final video at `/demo`.

## Superteam form

- [x] Project title and description drafted.
- [x] Live MVP link.
- [x] Public repository link.
- [x] Technical documentation link.
- [x] Detailed TxLINE feedback.
- [x] AI assistance disclosure.
- [x] Confirm the final demo link after deployment.
- [ ] Add optional X link only if the entrant wants to publish one.

## Entrant-controlled final actions

These actions are tied to the entrant’s identity or legal acceptance and should
be performed by the human entrant:

- sign in to or create the Superteam account;
- complete any required profile or KYC fields;
- test the final product and video;
- confirm the listing requirements and legal checkbox;
- perform the final submission click.
